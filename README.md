# Mateify: Dashboard de gestión de pagos💸
## App web con la posibilidad de gestionar tus pagos, administrar tus ingresos y representar gráficamente (gracias a uishadcn/charts) tus gastos a lo largo del mes. Este README se actualizará el dia de mañana con una lista de todas las futuras features.

## Tecnologias usadas: Angular y TailwindCSS

### Proyecto personal y 100% cliente con el único objetivo de desplegar un poco de mis conocimientos en el framework de Angular, uno al que venia fichando hace ya tiempo. En este proyecto, aplico todos los conocimientos básicos que he aprendido en Angular los últimos dos meses. Como no es mi primer proyecto de Angular, varias buenas prácticas se aplicarán a este.
