# Mateify: Página de reseña e información de peliculas y series!
## App web que despliega gracias a [OMDB](https://www.omdbapi.com/) información de múltiples películas y series. El proyecto es sencillo, pero su objetivo es poder desplegar aquí todos mis conocimientos en Angular.

## Tecnologias usadas: Angular y TailwindCSS

### Proyecto personal y 100% lado cliente con el único objetivo de desplegar un poco de mis conocimientos en el framework de Angular, uno al que venia fichando hace ya tiempo. En este proyecto, aplico todos los conocimientos básicos que he aprendido en Angular los últimos dos meses. Como no es mi primer proyecto de Angular, buenas prácticas se aplicarán en este.
